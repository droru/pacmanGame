    We used the jQuery Plugin "howler" for setting the music during the game, "color picker" for choosing the desired colors for the dots on the board, and "jQuery validation" for validation the user's inputs in the login/register forms by a specified regex expresion.
    We didn't use any prepared template, we preferred to experience with it ourselves, but we used a lot the google to find the desired fabulous designs, animations, and parts of code that can help us,
    for example, to draw the ghosts we found a helper function that actually draws the ghost on the canvas (without any images) including drawing the ghost's eyes according to its moving direction.
    Our difficulties:
    *Sometimes it was difficult to debug the code because of the intervals that accrued every 250 ms.
    *Setting the movement of the ghosts was a little bit tricky for us.
